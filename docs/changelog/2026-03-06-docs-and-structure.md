# 2026-03-06: Docs and Structure Baseline

## Summary

- Introduced a canonical `docs/` taxonomy.
- Moved setup, architecture, runbook, and roadmap docs out of repo root.
- Archived transient AI-generated status/plan outputs to `docs/archive/ai-sessions/`.
- Added canonical feature docs for boxing API, UI, and password reset.

## Rationale

The repository had accumulated many overlapping plan/status markdown files and mixed concerns across app/runtime content and documentation artifacts. This baseline creates a predictable structure for contributors and keeps non-runtime docs out of the app bundle.
